# Location Mocking Guide - Android

The Android Emulator provides built-in tools to mock GPS location for testing location-based features.

1. **Open Extended Controls**
   - Click the **three dots (â‹®)** on the emulator toolbar
   - Or press `Ctrl + Shift + P` (Windows/Linux) or `Cmd + Shift + P` (macOS)

2. **Navigate to Location**
   - Select **Location** from the left sidebar

3. **Set Location**
   - Search for a location:
     - Latitude and longitude coordinates
     - Address search
     - Location name search
   - Click **Save Point** to add the location to your 'saved points' list
   - To use a saved location, select it from your saved points list and click **Set Location**
